# 💰 PesaTrack

> **Track every shilling. Build better money habits.**

PesaTrack is a modern personal finance application that helps users track their income, expenses, and spending habits through a clean, intuitive interface.

This project began as a simple **Student Expense Tracker** built while learning JavaScript fundamentals. Rather than abandoning the original project, it is being refactored and expanded into a production-inspired application that follows better software engineering practices and modern project architecture.

The goal is not only to build a useful application, but also to document the journey of growing from a beginner project into a scalable full-stack product.

# 🎯 Project Goals

PesaTrack aims to help users:

- Record income and expenses
- Monitor spending habits
- View financial summaries
- Understand spending by category
- Build healthier financial habits

As development progresses, the application will evolve into a complete personal finance platform.
